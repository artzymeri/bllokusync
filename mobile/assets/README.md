# Adding the Logo to Mobile App

## Steps to add the logo:

1. Copy the logo file from the frontend:
   - Source: `/Users/artz./Desktop/Private/bllokusync/frontend/public/app_logo.png`
   - Destination: `/Users/artz./Desktop/Private/bllokusync/mobile/assets/logo.png`

2. You can do this by running in terminal:
   ```bash
   cp /Users/artz./Desktop/Private/bllokusync/frontend/public/app_logo.png /Users/artz./Desktop/Private/bllokusync/mobile/assets/logo.png
   ```

3. The logo will automatically appear in the login screen once copied.

## Alternative: Use the provided logo image
Save the BllokuSync logo image you provided as `logo.png` in the `/Users/artz./Desktop/Private/bllokusync/mobile/assets/` folder.
